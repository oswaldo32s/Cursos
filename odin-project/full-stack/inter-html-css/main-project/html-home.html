Tables

Probably something I might not use that often, so it is very important to keep this notes saved

Main table elements

  • <table> - This will mark the beginning and end of the table.
  • <thead, tbody, tfoot> - This is pure semantic.
  • <tr> - This will create a new row on the table.
  • <th> - This will create a header title.
  • <td> - This will create a regular data cell in the table.

You can add colspan="" or rowspan="" to make a th or td take more than 1 space.

Table Styling

For a better look on the tables it is important to add the following styling:

table, th, td {
     border: 1px solid white;
     border-collapse: collapse;
}
